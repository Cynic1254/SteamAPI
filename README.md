# Steam API for Unreal Engine
An Unreal Engine plugin aiming to modernize the Steam API
## Features

- Full Steam input API support, bind input directly to steam actions
- Button Display, Widget for automatically displaying the button the user should press to perform the action
- in-depth debugging, Use the visual debugger to see what steam buttons are currently pressed and what action set is active

## Installing
Simply place the folder inside the plugin folder, when prompted to rebuild select "yes"

For an in-depth guide on how to use this plugin, refer to the [wiki](https://github.com/Cynic1254/SteamAPI/wiki)
